**Data Cleaning & Bike Sales Dashboard Project**

#Project Overview
This repository includes two separate but connected Excel-based projects:
Data Cleaning – Performed in a dedicated workbook using a raw tutorial dataset.
Bike Sales Dashboard – Created using a separate bike buyers dataset to visualize sales insights.
Both parts demonstrate strong skills in Excel analytics, transformation, and dashboard design.

1. Data Cleaning.xlsx
This file is fully dedicated to data cleaning tasks.
It includes:
Raw dataset (tutorial-based)
Cleaning steps such as:
Removing duplicates
Standardizing formatting
Fixing inconsistent text
Handling missing values
Creating clean analytical fields
This file focuses exclusively on transforming messy raw data into a high-quality, analysis-ready dataset.

2. Bike_Sales_Dashboard.xlsx
This file contains the bike buyers dataset and the complete dashboard workflow.
Sheets included:
bike_buyers — Main dataset with demographics + bike purchase status
Working Sheet — Intermediate cleaning/prep specific to dashboard creation
Pivot Table — Summaries used in visualizations
Dashboard — Interactive Excel dashboard showing:
Buyer demographics
Income vs. purchase behavior
Commute patterns
Age-group purchase trends
Regional sales insights
Key metrics (KPIs)
This file focuses on exploratory analysis and visualization, not on general data cleaning.

Dashboard Highlights
The Excel dashboard visualizes:
Purchase trends by age, gender, and income 
Sales breakdown by region
Commute distance effect on bike purchasing
Customer distribution across education, occupation, and marital status
Dynamic KPI summaries using pivot tables and slicers
